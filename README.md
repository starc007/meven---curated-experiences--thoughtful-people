# Luma Landing Page

A premium landing page for Luma, a curated application-based event platform that focuses on quality experiences with thoughtful people.

## Framework

React 19 + Vite 6 with Tailwind CSS and Framer Motion

## Preview

Configured to run on `0.0.0.0:8080` for k8s HTTPRoute access.

## Development

```bash
npm install
npm run dev
```

Server will be available at:
- Local: http://localhost:8080
- Network: http://0.0.0.0:8080

## Build

```bash
npm run build
npm run preview
```

## Theme

The design follows a calm, premium, and editorial aesthetic with a soft neutral color palette based on cream, sand, and charcoal tones. Typography uses Cormorant Garamond for display text and Inter for body text.

## Features

- Fully responsive design (mobile, tablet, desktop)
- Smooth scroll animations with Framer Motion
- Tailwind CSS for styling
- Modern React patterns and hooks
- Subtle animations that enhance rather than distract
- Accessible and semantic HTML

## Structure

- **Hero Section** - Bold statement with clear value proposition
- **What Luma Is** - Short statements about the platform's core features
- **For Hosts** - Benefits and features for experience hosts
- **For Guests** - Benefits and features for experience attendees
- **How It Works** - Simple 3-step process explanation
- **Trust & Reputation** - Focus on the platform's reputation system
- **Call to Action** - Email signup for requesting access
- **Footer** - Navigation, links, and copyright information
